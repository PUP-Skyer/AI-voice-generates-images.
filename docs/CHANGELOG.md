# 📋 更新日志 (CHANGELOG)

## v1.2.0 (2026-06-14) - 语音输入版

### 🎤 新增功能

- **语音输入**：在输入栏新增麦克风按钮，支持语音转文字输入
- **实时转写**：语音识别结果实时填充到输入框
- **追加模式**：支持在已有文本基础上继续语音输入
- **录音动画**：按钮录音时显示红色脉冲光晕效果
- **状态提示**：按钮上方显示"正在聆听..."浮层
- **错误处理**：针对6种错误场景提供友好提示
- **自动重启**：Chrome静默超时后自动恢复监听

### 🔧 技术改进

- 集成 Web Speech API（SpeechRecognition）
- 添加 CSS 动画（脉冲光晕、状态提示）
- 优化错误处理逻辑（网络/权限/设备）
- 发送消息时自动停止录音

### 📝 文档

- 更新 README 文档，聚焦语音输入功能
- 添加演示视频脚本
- 添加贡献指南

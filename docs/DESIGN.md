# 五哥生图语音版 - 设计文档

> 版本：v1.2 | 最后更新：2026-06-15

---

## 1. 项目概述

### 1.1 项目背景

五哥生图是一款基于五哥科技API的AI图片生成工具。本项目在原有基础上，新增**语音输入功能**，用户可以通过说话代替打字来输入图片描述词。

### 1.2 技术栈

| 技术 | 用途 |
|------|------|
| HTML5 + Tailwind CSS | UI |
| JavaScript (ES6+) | 交互逻辑 |
| Web Speech API | 语音识别 |
| IndexedDB + localStorage | 数据持久化 |

## 2. 系统架构

### 2.1 整体架构

```
┌──────────────────────────────────────────────┐
│             五哥生图 - 语音输入版               │
├──────────────────────────────────────────────┤
│  UI层: 左侧栏 | 对话区 | 输入区 | 画廊 | 轻箱  │
├──────────────────────────────────────────────┤
│  逻辑层:                                      │
│  ┌───────────┐ ┌──────────┐ ┌────────────┐  │
│  │ 语音输入   │ │ 对话管理  │ │ API调用引擎 │  │
│  │ 模块      │ │ 模块     │ │            │  │
│  └───────────┘ └──────────┘ └────────────┘  │
│  ┌───────────┐ ┌──────────┐ ┌────────────┐  │
│  │ 图片缓存   │ │ 画廊管理  │ │ 图片操作   │  │

```

### 2.2 核心数据流

```
用户语音 → Web Speech API → 文字 → 输入框 → 发送 → 五哥API → 图片
```

## 3. 语音输入模块设计

### 3.1 模块接口

| 函数 | 说明 |
|------|------|
| checkVoiceSupport() | 检测浏览器支持 |
| initVoiceRecognition() | 初始化语音识别 |
| toggleVoiceInput() | 切换录音/停止 |
| startRecording() | 开始录音 |
| stopRecording() | 停止录音 |
| showVoiceError(type) | 显示错误提示 |

### 3.2 识别流程

```
点击按钮 → startRecording()
  ├── 保存已有文本
  ├── 调用 recognition.start()
  ├── 按钮变红+动画
  └── 显示"正在聆听..."
说话 → 实时转写到输入框
点击停止 → stopRecording()
```

### 3.3 错误处理

| 错误 | 处理 |
|------|------|
| network | 重试1次，连续失败2次停止 |
| not-allowed | 停止，提示授权 |
| no-speech | 静默重启 |
| audio-capture | 停止，提示检查设备 |

## 4. 原创功能清单
